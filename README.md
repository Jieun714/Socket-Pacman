소켓 통신을 이용한 멀티 팩맨 게임

## ⭐ Project Introduction
<h3> 기능 </h3>
- 2 Players game – 많은 쿠키를 먹은 팩맨이 승리
- 점수 관리 기능
- 채팅 관리 기능
<br>

<h3> 게임 설명 </h3>
- 2인 플레이 게임으로써 두 명이 들어와 있어야 게임 진행이 가능하다.
- 고스트를 피해 쿠키를 많이 먹는 팩맨이 승리하는 게임이다.
- 고스트와 부딪치면 팩맨은 죽게 된다.
